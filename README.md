# Sql Performance Analysis - order system

Projekt przedstawia analizę i optymalizację zapytań SQL w PosgreSQL.

## Zakres:
- analiza wolnych zapytań
- identyfikacja problemów wydajnościowych
- optymaizacja poprzez indeksy

## Technologie:
  - PostgreSQL
  - SQL

## Dodanie indeksow:
Po dodaniu indeksów czas wykonania zapytania uległ znaczącej poprawie.
